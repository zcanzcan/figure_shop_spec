\# 프로젝트명: 하이엔드 피규어 콜렉터 샵 (Real-world MVP)



\## 핵심 데이터 및 진열

• 고유 시리즈별 카테고리 분류 (Marvel, Anime, Game)

• 상세 정보 포함: 스케일(1/6, 1/4), 제조사, 에디션 정보

• 실시간 재고 상태값 반영 (품절 임박, 품절 표시 로직)

• 검색 및 필터링: 시리즈별, 가격대별 정렬 기능



\## 실전형 기능

• 장바구니: 수량 실시간 조절, 개별/전체 삭제, 총 결제 금액 자동 합산

• 주문서 섹션: 수령인 성함, 연락처, 배송 주소 입력 필드 구현

• 결제 연동: 포트원(Portone) API 기반 카카오페이/카드 결제 연동

• 사후 처리: 결제 성공 시 고유 주문번호 생성 및 로컬 스토리지 주문 내역 저장



\## 사용자 경험(UX)

• 모바일 환경 최적화 반응형 그리드 레이아웃

• 결제 버튼 클릭 시 유효성 검사 (주소지 누락 확인 등)

• 결제 완료 후 '구매 성공' 결과 페이지 및 주문 요약 노출
